# Helm Filter Plugin

Helm plugin to filter out template files

## Install

Fetch the latest binary release of helm-values and install it:
 
```sh
$ helm plugin install https://github.com/softleader/helm-filter
```

## Usage
 
```sh
$ helm filter [flags] CHART
```
